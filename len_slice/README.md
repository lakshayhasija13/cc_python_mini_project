# Len's Slice

A Python script that manages a pizza shop's toppings and prices using lists.

## Description

This project builds a combined list of `[price, topping]` pairs for a pizza menu, sorts them by price, identifies the cheapest and priciest options, removes the most expensive item, adds a new topping, and slices out the three cheapest pizzas.

## Concepts Used

- Lists and nested lists
- `list.sort()`, `list.count()`, `list.pop()`, `list.append()`
- List slicing
- `len()` built-in
- f-strings and `print()` statements
