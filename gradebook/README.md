# Gradebook

A Python script that manages a student gradebook using nested lists.

## Description

This project creates a gradebook as a list of `[subject, grade]` pairs, demonstrates common list operations (appending, updating, removing), and combines the current semester's gradebook with the previous semester's records.

## Concepts Used

- Nested lists
- `list.append()`, `list.remove()`
- List indexing and mutation
- List concatenation (`+`)
- `print()` statements

## Sample Output

```
[['politics', 80], ['latin', 96], ['dance', 97], ['architecture', 65], ['physics', 98], ['calculus', 97], ['poetry', 'Pass'], ['history', 88], ['computer science', 100], ['visual arts', 98]]
```

## How to Run

```bash
python code.py
```
